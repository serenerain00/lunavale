# Roadmap

## Phase 0 — Story and World Foundation

- Add the first three story folders
- Normalize story IDs
- Define shared characters
- Define initial environments
- Inventory scenes and images
- Identify public and premium content
- Select first world-map concept
- Define motion tokens
- Select technical vendors

## Phase 1 — World Prototype

Build one complete vertical slice.

Recommended environment: Lakehouse.

Include:

- World entry
- Lakehouse environment
- Two or three areas
- Several objects
- One public memory
- One locked premium discovery
- GSAP environment transition
- Reduced-motion fallback
- Mobile adaptation
- Direct URLs

The purpose is to validate the experience before building every environment.

## Priority Rule

Build a feature now only when it strengthens at least one of:

1. Story understanding
2. Exploration
3. Watching
4. Audience ownership
5. Revenue
6. Professional presentation
