# Brand and Tone

## Essence

**Intimate. Cinematic. Mature. Emotionally dangerous.**

## Voice

Use concise, evocative language.

Preferred:

- Enter the world
- Return to the lakehouse
- A memory is waiting
- This room is locked
- Continue the story
- Enter the Vault

Avoid:

- Generic gaming language
- Loud sales language
- Corporate AI terminology
- Excessive explanation
- Cheap tabloid phrasing

## Product Language

Use:

- World
- Environment
- Place
- Memory
- Story
- Scene
- Chapter
- Artifact
- Journal
- Collection
- Vault
- Private
- Unreleased
- Extended cut

Avoid making everything sound like a technical asset or content post.
