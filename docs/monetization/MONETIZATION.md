# Monetization

## Core Concept

Membership is access to deeper parts of the world.

The user is not simply paying for a larger video list. They are receiving keys to private spaces, memories, artifacts, and story material.

## Free Access

- World entry
- Selected environments
- Public objects
- Trailer
- Public scenes
- Character introductions
- Story overview
- Selected behind-the-scenes content

## Free Account

- Saved discoveries
- Continue exploring
- Watch progress
- Selected account-only memories
- Release notifications

## Vault Membership

Potential benefits:

- Locked rooms
- Full scene library
- Early releases
- Mature scenes
- Extended cuts
- Alternate edits
- Journals
- Private character artifacts
- Creator commentary
- Scripts
- Production diaries
- Unreleased stills
- Member-only environment variants

## Higher Supporter Tier

Potential benefits:

- Everything in Vault Membership
- Monthly production breakdown
- Supporter credits
- Creator Q&A
- Selected high-resolution material
- Advanced workflow breakdowns

Avoid unsustainable personalized promises.

## One-Time Purchases

- Chapter bundles
- Full film releases
- Visual books
- Script collections
- Character art
- Production guides
- Story-bible excerpts

## Conversion Moments

- A locked door or object
- End of a free memory
- Story cliffhanger
- Character timeline reveal
- Alternate-cut preview
- New environment opening
- Member-only night or weather variant

## Ethical Rules

- No fake scarcity
- No deceptive billing
- Clear cancellation
- No shame-based messaging
- No repeated intrusive interruption
- Public content must still feel meaningful
