# Motion Language

## Motion Philosophy

Nothing should move merely because it can.

Motion must communicate one of the following:

- Travel
- Focus
- Memory
- Emotional change
- Unlocking
- Discovery
- Continuity
- Hierarchy
- Feedback

## Camera Vocabulary

### Slow Push

Meaning:

- Emotional attention
- Growing intimacy
- Curiosity
- Unease

### Pull Back

Meaning:

- Emotional distance
- Context
- Leaving a memory
- Returning to the world

### Pan

Meaning:

- Looking across a place
- Revealing nearby options
- Following a character or object

### Dolly or Track

Meaning:

- Entering a location
- Moving through an environment
- Crossing from one area to another

### Orbit

Meaning:

- Examining an important object
- Revealing multiple sides of a relationship
- Heightened attention

Use sparingly.

### Rack-Focus Simulation

Meaning:

- Shifting attention between foreground and background
- Revealing that an object contains a memory
- Connecting present and past

### Fade Through Black

Meaning:

- Major location change
- Emotional break
- Time transition
- Memory boundary

## Emotional Timing

Josh-related motion may feel:

- Controlled
- Heavy
- Magnetic
- Slightly constricting

Tyson-related motion may feel:

- Grounded
- Natural
- Warm
- Responsive

Luna-related motion may feel:

- Reflective
- Hesitant
- Layered
- Pulled between stillness and movement

These are creative tendencies, not rigid rules.

## UI Feedback

Buttons, locks, map markers, and cards should use subtle motion.

Avoid:

- Bouncy game UI
- Constant pulsing
- Exaggerated elastic easing
- Rapid zooming
- Spinning indicators without purpose
