# Security and Content Controls

## Core Rules

- Secrets remain server-side.
- Premium authorization is server-side.
- Verify billing webhooks.
- Use least privilege.
- Validate content data.
- Protect admin routes.
- Avoid permanent public premium URLs.
- Do not store payment card details.
- Collect minimal personal data.

## Premium Media

Use:

- Signed playback where available
- Entitlement checks
- Session validation
- Rate limiting
- Nonpublic storage
- Clear terms
- Optional watermarking for selected downloads

Perfect piracy prevention is not possible. Use reasonable deterrence.

## Mature Content

Support ratings:

- General
- Teen
- Mature
- Restricted 18+

Public browsing should remain tasteful.

## Rights Checklist

Before release confirm rights for:

- Music
- Fonts
- Stock media
- Likeness
- Visual assets
- Downloads
- Brand references
